# vegReclass_WBI
This SpaDES module reclassifies outputs from the Biomass_borealDataPrep and Biomass_speciesData SpaDES modules into specific forested classes for Bird Conservation Regions (BCR) for a given study area within the BCR. So far, a set of rules for BCR 6 in Alberta and British Columbia has been implemented. 

Also included is makeLandCoverClassRaster.R, which reclassifies LCC10 2010 land cover classes to a reduced number of classes for a given study area.
